### 🚀 Projetos em Destaque (Business Cases)

Aqui estão os principais cases reais onde apliquei inteligência comercial e regras de negócio para resolver desafios de faturamento, capacidade e processos operacionais:

#### 🛒 [Lotus Conveniência - Business Intelligence & Analytics](https://github.com/DivisionHs/lotus-conveniencia-sales-analytics)
* **Escopo:** Solução de BI focada no setor de varejo/conveniência para mapeamento de faturamento corporativo, diagnóstico de rotatividade de estoque e auditoria de processos.
* **Destaque Técnico:** Modelagem e tratamento de dados no Power BI alinhados com regras de negócio corporativas. Uso de script Python externo como ferramenta de suporte para criar um fluxo ágil de anonimização e mascaramento de dados comerciais sensíveis, garantindo a segurança do portfólio.

#### 💈 [Barbearia Jhow Men - Analytics & CRM Strategy](https://github.com/DivisionHs/barbearia-jhow-men-consulting)
* **Escopo:** Muito mais do que um desenvolvimento técnico ou visualização isolada em dashboards, este projeto consolidou-se como uma **parceria estratégica de consultoria ativa** para o negócio. Utilizando dados reais extraídos do ecossistema transacional da empresa, atuo diretamente no diagnóstico de ociosidade operacional, análise detalhada de capacidade produtiva por turnos e na estruturação de uma régua de saúde de clientes (Régua de Churn/CRM) para direcionar ações comerciais preventivas e de retenção.
* **Destaque Técnico:** Domínio avançado em fórmulas DAX para regras de negócios (cálculo de tempos produtivos reais vs. disponíveis) e modelagem dimensional clássica (*Star Schema*). Uso de automação em Python focado em mascaramento alfanumérico seguro de contatos para viabilizar a publicação do case.
